# RegionCascader 获取选择内容示例

本文说明如何获取 `RegionCascader` 选中的编码、文本和完整信息。

## 通过 v-model 获取编码

```vue
<template>
  <RegionCascader v-model="region" @change="handleChange" />
</template>

<script setup>
import { ref } from 'vue'

const region = ref([])

function handleChange(value) {
  // value 示例：[110000, 110100, 110101]
  console.log(value)
}
</script>
```

## 通过 change-with-labels 获取编码和文本

```vue
<template>
  <RegionCascader
    v-model="form.region"
    @change-with-labels="handleRegionChange"
  />
</template>

<script setup>
import { reactive } from 'vue'

const form = reactive({
  region: [],
  regionText: ''
})

function handleRegionChange(data) {
  form.region = data.values
  form.regionText = data.labels.join('/')
}
</script>
```

`data` 结构：

```js
{
  values: [110000, 110100, 110101],
  labels: ['北京市', '北京市', '东城区']
}
```

## 通过组件方法读取完整信息

```vue
<template>
  <RegionCascader ref="regionRef" v-model="region" />
  <el-button @click="getInfo">获取完整信息</el-button>
</template>

<script setup>
import { ref } from 'vue'

const regionRef = ref()
const region = ref([])

function getInfo() {
  const info = regionRef.value.getSelectedInfo()
  // info: { values: Array, labels: Array, text: String }
  console.log(info)
}
</script>
```

## 表单提交示例

```vue
<template>
  <el-form :model="form" ref="formRef" label-width="100px">
    <el-form-item label="项目名称" prop="projectName">
      <el-input v-model="form.projectName" />
    </el-form-item>

    <el-form-item label="所在区域" prop="region">
      <RegionCascader
        v-model="form.region"
        @change-with-labels="handleRegionChange"
      />
    </el-form-item>

    <el-button type="primary" @click="submit">提交</el-button>
  </el-form>
</template>

<script setup>
import { reactive, ref } from 'vue'

const formRef = ref()
const form = reactive({
  projectName: '',
  region: [],
  regionText: ''
})

function handleRegionChange(data) {
  form.regionText = data.labels.join('/')
}

function submit() {
  const payload = {
    projectName: form.projectName,
    region: form.region,
    regionText: form.regionText
  }
  console.log(payload)
}
</script>
```

## 回显数据

后端保存编码数组后，编辑页直接赋值给 `v-model`：

```js
region.value = [110000, 110100, 110101]
```

如果需要立即读取文本，可在组件渲染后调用：

```js
regionRef.value.setValue([110000, 110100, 110101])
const info = regionRef.value.getSelectedInfo()
```
